Schnellhilfe

  • git remote -v – zeigt die gespeicherten Verbindungen zu Online-Archiven (Remotes).
  • git reset --hard HEAD~1 – macht den letzten Commit rückgängig (Reset = zurücksetzen).
  • tar -czf backup.tar.gz ordner/ – erstellt ein komprimiertes Archiv eines Ordners (Backup).
