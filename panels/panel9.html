Befehlsreferenz

  • git cherry-pick COMMIT – holt einen einzelnen Commit in deinen Verlauf (Cherry-Pick = gezielte Übernahme).
  • tail -f datei.log – zeigt ständig neue Zeilen einer Logdatei an (tail = Ende der Datei).
  • chmod +x script.sh – macht eine Datei ausführbar (chmod = Rechte ändern).
